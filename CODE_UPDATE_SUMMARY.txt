******************************************************************************
 * HOW TO USE THESE CODES:
 ******************************************************************************

For 8051/P89V51RD2 (Keil):
1. Open Keil µVision
2. Create new project, select P89V51RD2
3. Add .c or .asm file
4. Build project (F7)
5. Debug and test (Ctrl+F5)

For PIC18F4550 (MPLAB X):
1. Open MPLAB X IDE
2. Create project for PIC18F4550
3. Select XC8 compiler
4. Add .c file to Sources
5. Build (Clean and Build)
6. Program using PICkit
